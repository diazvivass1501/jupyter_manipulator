----      Toolbox de cinemática de manipuladores    --------
----      Versión corta                             --------
----      Compatible con Matlab R2016b o superior   --------

Esta carpeta contiene algunas funciones empleadas para el estudio de la cinemática de robots manipuladores, así como funciones para representar en 3D los diagramas cinemáticos de manipuladores.

A diferencia del toolbox de Peter Corke, la representación no se hace de forma automática a través de la representación Denavit Hartenberg, sino que las transformaciones homogéneas son a gusto del usuario y por lo tanto la representación la construye el usuario.

Se proporcionan un par de ejemplos para usar este Toolbox.

---- Dependencias y comentarios -----

Para poder establecer más de un colormap en objetos de tipo superficie se requiere el toolbox freezeColors de John Iversen (2005-10).

Para la importación de archivos STL se requieren las funciones "stlread" y "patchslim" de Francis Esmonde-White (May 2010).

---  Instalación  ----

Agregar al path de Matlab esta carpeta (tambien el path de cada toolbox auxiliar).

---- Manipulator kinematics toolbox          --------
----      Short version                      --------
---- Compatible with Matlab R2016b or higher --------

This folder contains some functions used to study the kinematics of manipulator robots, as well as functions to represent the kinematic diagrams of manipulators in 3D.

Unlike Peter Corke's toolbox, the representation is not done automatically through the Denavit Hartenberg representation, but the homogeneous transformations are at the user's discretion and therefore the representation is built by the user.

A couple of examples are provided for using this Toolbox.

---- Complementary toolboxes and comments -----

To be able to establish more than one colormap on surface type objects, the freezeColors toolbox by John Iversen (2005-10) is required.

For the import of STL files the functions "stlread" and "patchslim" from Francis Esmonde-White (May 2010) are required.

--- Installation ----

Add this folder to the Matlab path (also the path of each auxiliary toolbox).
